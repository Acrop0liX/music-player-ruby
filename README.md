# Music Player

A command-line music library application built in Ruby.

## Features
- Read albums from a text file
- Display all albums or filter by genre
- Add new albums with track details
- Edit existing album details
- Save albums back to file
- Input validation for duration format (M:SS)
- Duplicate album detection
- Input validation using input_functions.rb

## How to run
Requires Ruby. Run the following command (eg. VSCode, Terminal):

*ruby simple_menu_rev2.rb*
